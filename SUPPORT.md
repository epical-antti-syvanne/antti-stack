# Support

Antti Stack is an open-source project, not a managed service wearing a lanyard.

## Where To Ask

Use GitHub issues for:

- bugs
- documentation gaps
- feature requests
- unclear CLI behavior
- examples that drift into LinkedIn soup

Use pull requests for:

- fixes
- better examples
- new detectors
- tests
- docs improvements

## Before Opening An Issue

Please include:

- command used
- input text, if safe to share
- expected output
- actual output
- operating system and Node version when relevant

Do not include confidential customer data, secrets, internal project names, or anything from a spreadsheet named like it has seen things.

## Tone

Mock systems, not people.

The project is dry, but the maintainers should not have to become damp.
